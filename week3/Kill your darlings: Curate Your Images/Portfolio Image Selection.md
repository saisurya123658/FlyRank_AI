# Curate Your Images – Portfolio Image Selection

## Final Image Set (Keepers)

| Portfolio Section | Image                                                                                                    | Source                           | Reason                                                                                |
| ----------------- | -------------------------------------------------------------------------------------------------------- | -------------------------------- | ------------------------------------------------------------------------------------- |
| Hero Section      | Clean blue gradient abstract background                                                                  | AI Generated                     | Creates a modern, professional first impression without distracting from the content. |
| About Me          | Professional headshot                                                                                    | Real Photo                       | Personal branding should use an authentic photograph.                                 |
| Projects          | Screenshots of my completed projects (Temple Management System, Inventory Management System, AI Chatbot) | Real Screenshots                 | Demonstrates actual work and provides credibility.                                    |
| Skills            | Minimal line icons for programming languages and tools                                                   | AI Generated / Open-source icons | Maintains a consistent visual style and improves readability.                         |
| Experience        | Company logos                                                                                            | Official Logos                   | Represents organizations accurately.                                                  |
| Certifications    | Screenshots or PDF previews of certificates                                                              | Real Captures                    | Verifies achievements with authentic evidence.                                        |
| Contact           | Simple mail, GitHub, and LinkedIn icons                                                                  | Open-source icons                | Keeps the contact section clean and recognizable.                                     |

---

## Real Images vs AI Images

### Used Real Images For

* Professional profile photo
* Project screenshots
* Internship certificates
* Hackathon certificates
* GitHub repository screenshots
* Portfolio project demonstrations

### Used AI Images For

* Hero background texture
* Abstract decorative elements
* Minimal geometric illustrations
* Section dividers

---

## Rejected Image

I rejected an AI-generated futuristic developer workspace because it looked visually impressive but did not represent my actual work or portfolio. It drew unnecessary attention away from the projects and reduced the authenticity of the website.

---

## Style Consistency

All AI-generated visuals follow a minimal and modern design using soft blue gradients, clean geometric shapes, and subtle backgrounds that match the overall portfolio identity kit. Decorative visuals are intentionally understated so that the real project screenshots remain the primary focus.

---

## Design Decision

The portfolio prioritizes authenticity by using real screenshots and photographs wherever proof of work is required. AI-generated images are limited to decorative elements that improve visual appeal without replacing evidence of real skills or completed projects.
